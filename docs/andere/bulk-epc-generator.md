Een Desktop WPF-applicatie (.NET 9) ontworpen om EPC/SEPA-betaalketen QR-codes in bulk of per stuk te genereren vanuit een tekstbestand. De gegenereerde QR-codes bevatten extra transactie-informatie die duidelijk onder de code wordt geplaatst, ideaal voor facturatie.
